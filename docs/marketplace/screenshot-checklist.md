# Screenshot Checklist

Use safe demo data only. Do not show real schools, students, payments, credentials, SMTP values, or license keys.

## Public And Auth

- Landing page.
- Public result checker.
- Admin login.
- School login or workspace selection.

## Platform Admin

- Super Admin dashboard.
- School management.
- Feature/deployment mode visibility.
- License status.
- Guided updates.
- Backups.
- Demo sessions.
- Marketing pipeline where enabled.

## School Workspace

- School Admin dashboard.
- Student list and profile.
- Teacher assignment.
- Manual result entry.
- CSV result upload.
- Result review and publishing.
- CBT dashboard.
- Scratch-card workflow.
- Report card settings.

## Marketplace Rules

- Crop out browser bookmarks, local hostnames, and private directories.
- Use neutral placeholder emails and names.
- Do not show `.env`, logs, database dumps, API keys, or backup paths.
