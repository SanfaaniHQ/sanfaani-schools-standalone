# Live Demo Sales Flow

The marketplace live demo is a CodeCanyon-style preview for buyers who want to inspect Sanfaani Schools before purchase.

## Recommended Flow

1. Buyer visits `demo.sanfaani.net`.
2. Buyer opens `/demo/live`.
3. Buyer signs in with public demo credentials.
4. Buyer explores fake school data with destructive actions disabled.
5. Buyer chooses a purchase path.

## Purchase Paths

Standalone package:

- For technical buyers who can upload files, create a database, configure `.env`, and follow the cPanel-ready installation guide.
- Includes package/license/install support.
- Does not include SaaS hosting unless sold separately.

Done-for-you installation:

- For non-technical buyers.
- Sanfaani handles setup, environment configuration, and launch support.

SaaS:

- Buyer uses the hosted Sanfaani service.
- SaaS buyers do not get source code.

Do not claim real payment charging, billing automation, parent/student portals, or platform-super-admin demo access is complete unless those workflows are implemented and verified.
