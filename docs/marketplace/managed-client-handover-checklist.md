# Managed Client Handover Checklist

Managed client packages are for Sanfaani-operated or partner-operated deployments.

## Before Handover

- Confirm managed contract terms.
- Confirm hosting owner and access boundaries.
- Confirm license mode is `managed_contract`.
- Confirm deployment mode is `managed`.
- Confirm support escalation contacts.

## Operational Setup

- Validate package readiness.
- Configure hosting, domain, database, SMTP, and storage permissions.
- Create initial admin account and school profile.
- Record license validation status.
- Record update channel and backup policy.
- Create initial backup metadata and manual export.

## Handover Notes

- Managed backups are metadata and verification foundation unless separately contracted.
- Managed updates are package review and preflight foundation unless deployment automation is separately implemented.
- Restore execution is manual and not performed by the web UI.
- Marketplace API integrations and billing automation are planned, not implemented.
