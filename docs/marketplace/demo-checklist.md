# Demo Checklist

Use this checklist for marketplace demo, direct sales demo, and reseller demo preparation.

## Demo Setup

- Use demo-safe data only.
- Enable demo behavior only for controlled environments.
- Ensure demo users expire.
- Confirm passwords are not shown or stored in plain text.
- Disable destructive reset unless a safe demo-only reset workflow exists.

## Demo Walkthrough

- Landing page and buyer value proposition.
- Super Admin dashboard.
- Single-school installer preview.
- License status foundation.
- School Admin dashboard.
- Student and result workflow.
- CBT workflow.
- Public result checker.
- Guided onboarding.
- Update preflight and rollback plan foundation.
- Backup metadata and restore guidance foundation.

## Boundaries To Explain

- Billing/payment workflow is not fully implemented.
- Marketplace API integration is planned.
- Updates are metadata/preflight only in this foundation.
- Restore execution is manual and planned, not automated.
