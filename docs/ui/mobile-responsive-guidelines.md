# Mobile Responsive Guidelines

Sanfaani Schools must remain usable on phones for school admins, teachers, result officers, and support users.

## Layout

- Stack page header content and actions on mobile.
- Use `sm:` and `lg:` breakpoints for grids and action rows.
- Keep tables inside horizontally scrollable wrappers.
- Avoid fixed widths except for predictable controls.
- Use `min-w-0` and `overflow-wrap` where content can be long.

## Touch Targets

- Mobile buttons should be at least 44px tall.
- Keep related actions grouped with visible labels.
- Avoid dense icon-only controls unless the icon is already familiar and labeled for assistive tech.

## Text

- Do not use viewport-scaled font sizes.
- Keep labels short and allow wrapping.
- Avoid truncating critical identifiers unless the full value is available nearby.
