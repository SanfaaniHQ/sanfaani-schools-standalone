# Backup Before Release Checklist

- Confirm backup manager is enabled where appropriate.
- Confirm latest backup metadata exists.
- Confirm recent verified backup exists for update-preflight readiness when required.
- Confirm database, file, and config backup scopes are documented.
- Confirm shared-hosting manual cPanel/phpMyAdmin guidance is available.
- Confirm backup logs contain no secrets.
- Confirm restore plan is manual/planned and does not claim restore was performed.
