# Admissions For Schools Without A Website

No separate website is required. The standalone portal provides:

- `/admissions` for requirements and cycle availability
- `/admissions/apply` for secure application submission
- `/admissions/track` for private status checks

The portal is both the public front door and the admission source of truth in this setup. School staff manage the private workflow at `/admin/admissions`.

Before sharing the link, the school administrator should create an admission cycle, add requirements, verify the dates, and open the cycle. The form accepts only configured file types and sizes. Uploaded documents are private.

Applicants receive an application number and a tracking token. Tracking also supports the guardian phone number, but never exposes applicant lists or another family's record.

This gives a school an immediate online admission presence, consistent applicant records, and direct conversion into the student register without paying for a separate website first.
