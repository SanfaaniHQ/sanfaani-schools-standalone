# Student Manual

Student documentation is currently foundation-level.

## Current Status

Student role support exists in the role/onboarding foundation. Full student portal workflows are planned and should not be described as complete.

## Expected Future Scope

- View own dashboard.
- Review personal results where enabled.
- Access CBT or learning-related modules where assigned after the portal safely supports that access.

## Boundary Rule

Students must only access their own records.

## LMS Boundary

Student LMS viewing is deferred in the current foundation because safe user-to-student identity resolution is not implemented. Students should not see an LMS portal link until that relationship is implemented and tested.
