# Teacher Manual

Teacher access is based on role context, school assignment, and enabled features.

## Current Responsibilities

- Review assigned classes and subjects.
- Enter assessment scores where result entry is enabled.
- Use result submission and review workflows where assigned.
- Use guided onboarding steps for teacher orientation.

## Boundaries

Teachers must not access unrelated schools, unrelated classes, or unrelated student records.

## Planned

Additional teacher portal polish and advanced classroom workflows may be added later.
