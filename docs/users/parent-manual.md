# Parent Manual

Parent documentation is currently foundation-level.

## Current Status

Parent role support exists in the role/onboarding foundation. Full parent portal workflows are planned and should not be described as complete.

## Expected Future Scope

- View linked children.
- Review published results when permitted.
- Receive school communications.
- Access support or billing-related notices where enabled.

## Boundary Rule

Parents must only access records for linked children.
