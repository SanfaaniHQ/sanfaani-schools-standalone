# Report Branding Guide

Report/result branding hooks currently support:

- School or resolved branding logo fallback.
- Resolved primary color fallback for report headings.
- Escaped report footer text.

This foundation does not replace the existing report-card settings system. Existing school report settings remain the first place for academic report layout controls.

Future work:

- Report template theme presets.
- Branded PDF header/footer controls.
- Per-session report branding snapshots.
